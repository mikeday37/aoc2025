# aoc2025

Puzzle solutions for [Advent of Code 2025](https://adventofcode.com/2025) in Python.

## Notes

To run locally, you should download your own puzzle inputs and save them as `input.txt` under each appropriate day folder.  My known answers will not be checked by default, since they depend on my input.  The creator of Advent of Code [does not want puzzle inputs published](https://adventofcode.com/2025/about#faq_copying).

Once you have your own inputs, just open the `solve.py` for a day and hit F5 in VSCode to run just that day.  You'll need the Python extensions from Microsoft, which VSCode should offer automatically.
